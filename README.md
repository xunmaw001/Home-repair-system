# Home-repair-system
基于微信小程序的上门维修系统，为用户和维修师傅搭建便捷桥梁。用户无需下载 APP，在微信就能提交维修需求，描述故障、上传照片，还能查看师傅资质、评价后自主选择。可预约上门时间，线上完成支付。维修师傅能接收订单，安排工作行程。企业管理者可管理师傅信息、订单数据，统计分析业务情况。该系统减少中间环节，提高维修效率，让用户享受优质上门维修服务，也助力维修企业提升运营管理水平。
